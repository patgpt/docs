## Quick Start

1. Install the plugin:

```bash
npm install @wavefunction/tailwindcss-neumorphic
```

2. Add it to your Tailwind CSS configuration:

```js
// tailwind.config.js
module.exports = {
  plugins: [require('@wavefunction/tailwindcss-neumorphic')]
}
```

3. Start using the components:

```html
<button class="nm-button">Click me</button>

<div class="nm-card">
  <h2>Card Title</h2>
  <p>Card content...</p>
</div>
```

## Why Neumorphism?

Neumorphism combines the best of skeuomorphism and flat design:

- Soft, elegant aesthetics
- Clear visual hierarchy
- Tactile feedback
- Modern minimalism
- Subtle depth and dimensionality
