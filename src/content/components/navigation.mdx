# Navigation Components

A collection of neumorphic navigation components including navbar, sidebar, breadcrumbs, tabs, and pagination.

## Navbar

A responsive navigation bar with brand and menu items.

```html
<nav class="nm-navbar">
  <div class="nm-navbar-brand">Logo</div>
  <div class="nm-navbar-nav">

    <a href="#" class="nm-navbar-item">Pricing</a>

</nav>
```

## Interactive Examples

### Responsive Navbar with Mobile Menu

```jsx
function Navbar() {
  const [isOpen, setIsOpen] = React.useState(false);

  return (
    <nav class="nm-navbar">
      <div class="nm-navbar-brand">Logo</div>
      <div
        class={`nm-navbar-nav ${
          isOpen ? "flex" : "hidden"
        } md:flex flex-col md:flex-row absolute md:relative top-full left-0 right-0 md:top-auto bg-inherit`}
      >
        <a href="#" class="nm-navbar-item active">
          Home
        </a>
        <a href="#" class="nm-navbar-item">
          Features
        </a>
        <a href="#" class="nm-navbar-item">
          Pricing
        </a>
        <a href="#" class="nm-navbar-item">
          About
        </a>
      </div>
      <button
        class="nm-button-icon md:hidden"
        onClick={() => setIsOpen(!isOpen)}
      >
        <svg
          xmlns="http://www.w3.org/2000/svg"
          class="h-6 w-6"
          fill="none"
          viewBox="0 0 24 24"
          stroke="currentColor"
        >
          <path
            stroke-linecap="round"
            stroke-linejoin="round"
            stroke-width="2"
            d="M4 6h16M4 12h16m-7 6h7"
          />
        </svg>
      </button>
    </nav>
  );
}
```

### Interactive Tabs

```jsx
function Tabs() {
  const [activeTab, setActiveTab] = React.useState(0);
  const tabs = ["Overview", "Features", "Pricing", "FAQ"];

  return (
    <div>
      <div class="nm-tabs">
        {tabs.map((tab, index) => (
          <button
            key={index}
            class={`nm-tab ${activeTab === index ? "active" : ""}`}
            onClick={() => setActiveTab(index)}
          >
            {tab}
          </button>
        ))}
      </div>
      <div class="mt-4 p-4 nm-flat rounded-lg">
        {/* Tab content */}
        Content for {tabs[activeTab]}
      </div>
    </div>
  );
}
```

## Accessibility

- Use semantic HTML elements (`nav`, `button`, `a`)
- Include ARIA labels and roles where appropriate
- Ensure keyboard navigation works correctly
- Maintain sufficient color contrast
- Add skip links for keyboard users

## Best Practices

1. **Responsive Design**

   - Design mobile-first
   - Use appropriate breakpoints
   - Consider touch targets on mobile

2. **Navigation Structure**

   - Keep navigation hierarchy clear
   - Use active states to show current location
   - Provide clear feedback on interaction

3. **Performance**
   - Use CSS transitions sparingly
   - Consider lazy loading for complex navigation
   - Optimize SVG icons
