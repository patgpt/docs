## Helper Functions

### Shadow Generator

```js
// Generate box shadow
const generateShadow = (shadowValues, darkShadow, lightShadow) => {
  return shadowValues
    .map((value, index) => `${value} ${index === 0 ? darkShadow : lightShadow}`)
    .join(",\n");
};

// Usage
const shadow = generateShadow(
  ["6px 6px 12px", "-6px -6px 12px"],
  "var(--nm-shadow)",
  "var(--nm-highlight)"
);
```

### Color Variables Generator

```js
// Generate color variables
const generateColorVariables = (theme) => ({

// Generate dark mode color variables
const darkModeColorVariables = (theme) => ({

```

## Best Practices

1. **Shadow Usage**

   - Use appropriate shadow sizes based on element size
   - Consider depth hierarchy in your design
   - Maintain consistent shadow direction
   - Use transitions for interactive elements

2. **Color Management**

   - Keep color palette consistent
   - Test colors in both light and dark modes
   - Consider accessibility contrast ratios
   - Use CSS variables for dynamic theming

3. **Animation Performance**

   - Use hardware-accelerated properties
   - Avoid animating expensive properties
   - Keep animations subtle and purposeful
   - Test performance on mobile devices

4. **Responsive Design**
   - Adjust shadow sizes for different viewports
   - Consider touch targets on mobile
   - Test transitions on different devices
   - Optimize for different screen sizes

## Examples

### Interactive Card

```jsx
function InteractiveCard() {
  const [isPressed, setIsPressed] = React.useState(false);

  return (
    <div
      className={`p-6 rounded-lg transition-all duration-200 ${
        isPressed ? "nm-pressed" : "nm-flat hover:nm-convex"
      }`}
      onMouseDown={() => setIsPressed(true)}
      onMouseUp={() => setIsPressed(false)}
      onMouseLeave={() => setIsPressed(false)}
    >
      <h3 className="text-lg font-semibold mb-2">Interactive Card</h3>
      <p>Click me to see the pressed state!</p>
    </div>
  );
}
```
